Service registrationÂ¶

Services are registered as entry points for their ServiceDescriptor - i.e. the instance of ServiceDescriptor for installed service is returned by EntryPoint.load().

The default group for service registration is saturnin.service, but itâ€™s possible to install additional service discovery iterators.

Custom service iterator must be a generator that accepts optional uid in string format, and yileds EntryPoint instances. If uid is specified, it must return only EntryPoint only for service with given uid, otherwise it should return all services.

Note

Custom iterator can return objects that are not EntryPoint instances, but they MUST implement load() method that will return ServiceDescriptor instance.

Custom iterators must be registered as entry points in saturnin.service.iterator group.
